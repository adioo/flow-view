Layout
======

Basic layout module for Engine.

## Configuration

```json
{
    "client": {
        "title": "string",
        "notFound": "?",
        "locale": {
            "value": "de",
            "possible": ".*",
            "cookie": "_loc"
        }
    },
}
```

If `locale` value is a string, then it will represent the value of the locale value.
